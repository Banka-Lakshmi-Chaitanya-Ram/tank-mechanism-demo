# 🚀 Tank Simulation (Unity)

This is a simple **Tank Simulation project** built using Unity as part of my learning and practice.

---

## 🎮 What this project does

* Tank movement using keyboard input
* Smooth rotation based on player input
* Turret (tower) aiming using mouse position
* Shooting system (bullets fired from tank)
* Camera follow system
* Basic environment and collision setup

## 📦 How to run this project

### Option 1: Open in Unity

1. Download or clone the repository
2. Open Unity Hub
3. Click **Add Project**
4. Select this folder
5. Open the project and press ▶️ Play

---

### Option 2: Try the build (if available)

If you don’t have Unity installed, you can download the build (if provided in this repo) and run it directly.
